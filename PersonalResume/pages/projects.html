Yugioh Progressive: A Card Game Simulator

Yugioh Progressive is a simulator for the popular trading card game Yugioh that is currently still in early development. It was developed to offer a Yugioh play environment that was optimized for a limited format or a popular style of play called Progression. The game was developed with the Unity Gaming Development Platform and it integrates some of Unity's Gaming Services including anyonomous Authroization, Lobbies, and Relay. Also included in the platform is web API integration with the ygoprodeck API:

Unity Editor Scripting

Connor, after developing the alpha iteration of Yugioh Progressive was dissatisfied at the organizational processes that were used and were available by Unity. Through this issue, he grew interest in Unity's Editor scripting and developing tools for Unity game development. First, he made a small project, the Enum Creator Editor Window . This was an introduction to the editor scripting environment for Connor. It was short however, extremely important so that Connor could have a basis in this skill. Now, he works on a larger project, a Sound System manager that will help the game development process by organizing sounds into types and allowing for automatic creation of a Sound volume menu.
